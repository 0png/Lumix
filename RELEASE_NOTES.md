# Lumix v0.1.0-beta Release Notes

> **Release Date**: January 17, 2025  
> **Version**: 0.1.0-beta  
> **Platform**: Windows 10+ (x64)

---

## 🎉 首次 Beta 版本發布

這是 Lumix 的第一個公開測試版本，提供完整的 Minecraft 伺服器管理功能。

---

## ✨ 主要功能

### 伺服器管理
- **多核心支援** - 支援 Vanilla、Paper、Fabric、Forge 四種伺服器核心
- **一鍵建立** - 簡單的介面引導建立新伺服器
- **版本選擇** - 自動獲取最新的 Minecraft 版本列表
- **記憶體配置** - 彈性設定伺服器記憶體使用量

### Java 環境
- **自動偵測** - 自動掃描系統中已安裝的 Java 版本
- **版本管理** - 支援 Java 8 到最新版本
- **手動設定** - 可自訂 Java 執行檔路徑

### 即時控制
- **伺服器控制台** - 即時查看伺服器日誌輸出
- **指令輸入** - 直接在介面中發送伺服器指令
- **狀態監控** - 即時顯示伺服器運行狀態

### 屬性編輯
- **視覺化編輯器** - 透過 GUI 編輯 server.properties
- **常用設定** - 快速調整遊戲模式、難度、玩家數量等
- **進階選項** - 支援所有 server.properties 設定項目

### 使用者體驗
- **雙語支援** - English / 繁體中文
- **主題切換** - 淺色 / 深色 / 跟隨系統
- **響應式設計** - 適應不同視窗大小
- **流暢動畫** - 精心設計的過場動畫與互動回饋

---

## 🐛 已修正問題

### 關鍵修正
- **修正「新增伺服器」按鈕無法使用** - 補上遺漏的 `onCreateServer` prop 綁定
- **修正 Forge 安裝器路徑處理** - 解決伺服器名稱包含空格時安裝失敗的問題

### 改進項目
- 優化錯誤處理與使用者回饋
- 改善 UI/UX 一致性
- 強化檔案路徑處理邏輯

## 📦 安裝說明

### 系統需求
- **作業系統**: Windows 10 或更新版本
- **處理器**: x64 架構
- **記憶體**: 4GB RAM（最低需求）
- **Java**: Java 8 或更新版本（應用程式會自動偵測）

### 安裝步驟
1. 下載 `Lumix Setup Beta.exe`
2. 執行安裝程式
3. 選擇安裝路徑（或使用預設路徑）
4. 完成安裝後啟動 Lumix

### 首次使用
1. 啟動 Lumix 後會自動偵測 Java
2. 點擊「新增伺服器」按鈕建立第一個伺服器
3. 選擇核心類型、Minecraft 版本、設定記憶體
4. 同意 EULA 並建立伺服器
5. 等待下載完成後即可啟動

## ⚠️ 已知限制

### Beta 版本限制
- **僅支援 Windows** - macOS 和 Linux 版本尚未提供
- **Forge 新版本** - 部分 Forge 1.17+ 版本可能需要特殊啟動方式
- **網路依賴** - 建立伺服器時需要網路連線下載核心檔案

### 功能限制
- 不支援自動更新（需手動下載新版本）
- 不支援伺服器備份功能
- 不支援模組/插件管理
